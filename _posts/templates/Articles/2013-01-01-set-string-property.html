Set a string property

By default properties without a type are parsed as strings. You can also explicitly set invoke the string parser.

<wo.contentControl> <myVal constructor="string"> Hello wipeout! </myVal> <template> Binding to element value: <span data-bind="text: myVal"></span> </template> </wo.contentControl> <wo.view templateId="'set-string-property-script-1'" />
