Share parent scope

It is not always necessary to create a scope for each view model. A good example of this is in using the if control, where the scope of the if istself is never needed. You might also notice yourself writing bindings like data-bind="value: $parents[4].value" which is an indicator of un-needed scope.

You can share the parents binding scope by setting shareParentScope="true". in a template definition or in the constructor of a view model.

<wo.contentControl scope="'Parent scope'"> <template> <div> <wo.contentControl shareParentScope="true" scope="'Child scope 1'"> <template> Scope is: <span data-bind="text: scope"></span> </template> </wo.contentControl> </div> <div> <wo.contentControl scope="'Child scope 2'"> <template> Scope is: <span data-bind="text: scope"></span> </template> </wo.contentControl> </div> </template> </wo.contentControl> <wo.view templateId="'share-parent-scope-script-1'" />

By default, the shareParentScope property of a wo.if control is true.
