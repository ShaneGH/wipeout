Set the model from the parents model

In order to keep clean contextual boundaries, the model property of a child view model can be bound to a property of its parent's model

<wo.contentControl model="{ value: 'Hello wipeout!' }"> <template> <wo.contentControl model="$parent.model().value"> <template> Binding to a property on parent's model: <span data-bind="text: model"></span> </template> </wo.contentControl> </template> </wo.contentControl> <wo.view templateId="'bind-to-parents-model-script-1'" />

In this example there is a parent view model which has a child view model. The model of the child is bound to a property of the model of the parent. $parent points to the scope of the view model 1 level up
