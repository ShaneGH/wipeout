Bind to the property of an ancestor

There is a helper function which can find ancestors based on their proerties, type, depth or any other custom attribute you need.

// create a custom class to distinguish type window.myType = wo.contentControl.extend(function myType() { this._super(); }); <myType model="'Hello wipeout!'"> <template> <wo.contentControl model="$find(myType).model"> <template> Binding to a specific ancestor: <span data-bind="text: model"></span> </template> </wo.contentControl> </template> </myType> <wo.view templateId="'bind-to-ancestor-script-1'" />
