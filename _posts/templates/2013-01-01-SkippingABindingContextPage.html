It can be useful to skip a binding context if it not needed. For example, when using a wo.if view model, it is generally un necessary to need any of the propertes of this view model or to want to set one of its templateItems (TODO, link). In this case we can ignore the binding context of this element using the property.

The woInvisible property is a on the class and is false by defualt. It can be set to true in the view model constructor or in the template, but is not an observable and later changes will not reflect in the binding context.

The default value for the class is true.
