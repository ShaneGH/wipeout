Use the wo.if control to render content only if a condition is met. By default the woInvisible property of a wo.if is set to true, meaning that rendered children of the wo.if cannot bind to its properties

<span>Control 1:</span> <wo.if condition="true"> <!--The template if the condition is true--> <template> The condition is true </template> <!--The template if the condition is false--> <elseTemplate> The condition is false </elseTemplate> </wo.if> <br /> <span>Control 2:</span> <wo.if condition="false"> <!--The template if the condition is true--> <template> The condition is true </template> <!--The template if the condition is false--> <elseTemplate> The condition is false </elseTemplate> </wo.if>
