This function can be used in two ways.

1. Calling the constructor of the base class

Calling the constructor of the base class should always be the first line of code in a child class: var childView = wo.view.extend(function(templateId) { this._super(templateId); // custom logic.... });

2. Calling an overriden method in a base class

When overriding a base class method the _super method can be used to call overriden method // using childView from previous example and overriding onRendered function from wo.view childView.prototype.onRendered = function (oldNodes, newNodes) { this._super(oldNodes, newNodes); // custom logic.... };
