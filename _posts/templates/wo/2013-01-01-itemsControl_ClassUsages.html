The wo.itemsControl class is the wipeout method for dealing with collections. It has am itemsSource property, designed to accept an array of models. It also has an items property which will be auto populated with view models which correspond to the itemsSource models

<wo.itemsControl itemSource="[{firstName: 'John', lastName: 'Smith'}, {firstName: 'Mark', lastName: 'Walsh'}]"> <!--The template for the control as a whole--> <template> <h2>People:</h2> <!--The itemsControl binding renders each item into the parent element. It does not need a second argument, however knockout syntax requires it--> <ul data-bind="itemsControl: null"></ul> </template> <!--The template for each item in the itemSource--> <itemTemplate> <li> <label>First name:</label> <span data-bind="html: model().firstName"></span>, <label>Last name:</label> <span data-bind="html: model().lastName"></span> </li> </itemTemplate> </wo.itemsControl>
