The wo.contentControl class is a wo.view class with the ability to set its template as a string as well as an Id. For example:

<wo.contentControl model="{firstName: 'John', lastName: 'Smith'}"> <template> <label>First name:</label> <span data-bind="html: model().firstName"></span>, <label>Last name:</label> <span data-bind="html: model().lastName"></span> </template> </wo.contentControl>

The template and templateId properties are tightly bound, so that if the template property is set the templateId property will reflect this change, and vice versa
