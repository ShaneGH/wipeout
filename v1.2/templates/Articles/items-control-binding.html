The itemsControl binding

The items control binding tells an view model where to render its items. It doesn't need an argument and will throw an exception if not used by an itemsControl.

<wo.items-control item-source="['Hello wipeout 1', 'Hello wipeout 2']"> <template> <ul data-bind="itemsControl: null"></ul> </template> <item-template> <li> <span data-bind="text: model"></span> Index: <span data-bind="text: $index"></span> </li> </item-template> </wo.items-control> <wo.view templateId="'items-control-binding-script-1'" />
