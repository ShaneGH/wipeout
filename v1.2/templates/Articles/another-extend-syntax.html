An alternate extend syntax

There is another way to define a class using the method.

var MyClass = wo.object.extend({ constructor: function MyClass(balance) { this._super(); }, myMethod: function() { }, statics: { myStaticMethod: function() { } } });

Using this syntax

  • The constructor property will be the class constructor
  • Other items in the class definition will be appended to the class prototype
  • Items in the statics object will be appended to the constructor
Note: do not define instance (non static) properties in the object passed into the extend method. They will be appended to the prototype, effectively making them static. Put instance definitions inside the constructor function instead
