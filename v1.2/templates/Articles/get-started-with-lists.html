Using lists of items

Lists of items are at the core of all javascript frameworks, and wipeout is no different. Within wipeout, simple js objects cannot be rendered, and so need to be converted to wo.view objects first. You can do this with an

<wo.items-control itemSource="[{itemText: 'Item 1'}, {itemText: 'Item 2'}]"> <itemTemplate> <div data-bind="text: model().itemText"></div> </itemTemplate> </wo.items-control>
