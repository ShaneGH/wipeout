Create an anonymous template property

You may wish to define view models which can accept anonymous templates (or template strings) and convert them to a template id, such is with the set-template property of the control. You can do this with the templateProperty function.

Define the view model // build a view model with an anonymous template property (theTemplate) wo.viewModel("demoApp.myType", wo.content) .templateProperty("theTemplate") .build(); Use the view model <demo-app.my-type> <the-template> Hello Wipeout! </the-template> <set-template> <wo.view template-id="$this.theTemplateId"></wo.view> </set-template> </demo-app.my-type> Result: <wo.view template-id="'anonymous-t-prop-1'" />
