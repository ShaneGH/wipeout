Working with lists

The simplest way to handle lists is with the attribute. Foreach creates a new view model for each item in the attribute and adds the item as the model of the created view model.

<wo.content model="['Item 1', 'Item 2']"> <set-template> <ul wo-foreach="$model"> <li wo-content="$model + ', index: ' + $index.value"></li> </ul> </set-template> </wo.content> <wo.view template-id="'working-with-lists-example-1'"></wo.view>

The foreach attribure wraps the view model which can handle more complex situations.
