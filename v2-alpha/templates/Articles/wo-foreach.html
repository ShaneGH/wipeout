The Foreach attribute

wo-foreach="value"

data-wo-foreach="value"

The foreach attribute renders the content of an element only if the content of the attribue evaluates to true.

The foreach attribute wraps an , which can be used if you also need to define a more complex template.

<wo.content model="['Item 1', 'Item 2']"> <set-template> <ul wo-foreach="$model"> <li wo-content="$model + ', index: ' + $index.value"></li> </ul> </set-template> </wo.content> <wo.view template-id="'wo-foreach-attr'"></wo.view>
