Custom html attributes

You can also add custom html attributes to your application by using the wo.addHtmlAttribute method.

wo.addHtmlAttribute("clear-when-clicked", function () { return function (element, attribute, renderContext) { attribute.onElementEvent("click", renderContext, function () { element.value = ""; }); } }); <wo.content-control> <set-template> <input wo-clear-when-clicked /> </set-template> </wo.content-control> <wo.view template-id="'wo-custom-html-attribute-1'"></wo.view>
