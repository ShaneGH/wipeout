Set a date property

The date parser uses new Date(value) to parse dates. You can override this by .

<wo.content-control> <my-val constructor="date"> 2001/01/01 </my-val> <template> Binding to element value: <span data-bind="text: myVal"></span> </template> </wo.content-control> <wo.view templateId="'set-date-property-script-1'" />
