Share parent scope

It is not always necessary to create a scope for each view model. A good example of this is in using the control, where the scope of the if istself is never needed. You might also notice yourself writing bindings like data-bind="value: $parents[4].value" which is an indicator of un-needed scope.

You can share the parents binding scope by setting share-parent-scope="true". in a template definition or in the constructor of a view model.

<wo.content-control scope="'Parent scope'"> <template> <div> <wo.content-control share-parent-scope="true" scope="'Child scope 1'"> <template> Scope is: <span data-bind="text: scope"></span> </template> </wo.content-control> </div> <div> <wo.content-control scope="'Child scope 2'"> <template> Scope is: <span data-bind="text: scope"></span> </template> </wo.content-control> </div> </template> </wo.content-control> <wo.view templateId="'share-parent-scope-script-1'" />

By default, the shareParentScope property of a control is true.
