Using a parser

You can use a parser in one of two ways:

1. Setting a default parser for a view model property

When you are defining a view model you can set a default parser

wo.viewModel("myNamespace.myViewModel") .parser("myProperty", "string") .build();

2. Setting a parser in the template

You can override any default parsers by setting the parser in the template

<wo.view my-property--string="Hello wipeout!"></wo.view>

remember, most wipeout parsers also have short alieses, so rather than using my-property--string you can use the shorthand my-property--s
