Binding in code

You can also bind values in javascript using the function.

var objectToBindTo = ko.observable("Hello wipeout!"); var myView = new wo.view(); myView.bind("model", objectToBindTo, false); // Will log: "Hello wipeout!" console.log(myView.model); objectToBindTo("Goodbye wipeout!"); // Will log: "Goodbye wipeout!" console.log(myView.model);
