The if control

If/else functionality in a wipeout template is handled by the control.

<wo.if condition="true"> <set-template> The condition is true </set-template> <else-template> The condition is false </else-template> </wo.if>

One thing to note, is that defaults to true for an if control. This means that the if control cannot be seen by other view models. You can explicitly set this property to false in the template definition if this is not required.
