Functions and events

You can add custom functions to a viewModel which can then be called by user actions such as a mouse click.

// viewModel 1 var person = wo.viewModel("demoApp.person") .templateId("/person.html") .build(); // add a function to the person which will display the name person.displayFullName = function () { if (this.model) { alert(this.model.firstName + " " + this.model.lastName); } }; // viewModel 2 wo.viewModel("demoApp.name") // for this viewModel type we want // any changes to the "model" property // to reflect in the property it is bound to. // "tw" stands for "two way" .binding("model", "tw") .templateId("/name.html") .build();

At the URL "/person.html"

<div> First name: <demo-app.name model="$model.firstName"></demo-app.name> <br /> Last name: <demo-app.name model="$model.lastName"></demo-app.name> <br /> <button wo-click="$this.displayFullName">Display name</button> </div>

And at the URL "/person.html"

<input type="text" wo-value="$model" />

So the following element in html:

<demo-app.person model="{firstName: 'Robert', lastName: 'Paulson'}"></demo-app.person>
