The wipeout binding

The wipeout bindig is the root of a wipeout application and will only be used once. It accepts one parameter which is the root view model type.

Create a view model type window.rootApplication = wo.contentControl.extend(function rootApplication() { this._super(); this.template("Hello Wipeout!"); }); Create an application placeholder <div data-bind="wipeout: rootApplication" id="root"></div> Render the application var root = document.getElementById("root"); ko.applyBindings({}, root);
