Bind to a specific viewModel

When binding viewModel properties and event handlers, either using data-bind="" for html elements or setting the properties on viewModels, the current scope is the viewModel itself. So for instance, the setter property1="model().property1" will bind the property1 to the property1 property of the model of the current viewModel.
