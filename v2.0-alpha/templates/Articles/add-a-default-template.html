Add a default template

View models need templates to render. The templateId property can be set at any time during the lifecycle of the view model, and if you want the view model to have a default template you can set it here:

The most basic view model definition looks like this:

wo.viewModel("myNamespace.myViewModel") .template("/my-template.html", true) .build();

What this definition will do is:

  1. Ensure that all viewmodels created using the myViewModel class will have the templateId "/my-template.html". "/my-template.html" can be the id of a html element on the page, or the URL of the template html.
  2. The second argument true tells wipeout to attempt to load the template from the URL immediately, if it doesn't exist in the DOM. If this is not set, the template will not be loaded until the first time a myViewModel element is rendered.
