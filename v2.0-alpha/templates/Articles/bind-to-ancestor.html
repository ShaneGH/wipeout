Bind to the property of an ancestor

There is a helper function which can find ancestors based on their proerties, type, depth or any other custom attribute you need.

// create a custom class to distinguish type (inherit from contentControl) wo.viewModel("demoApp.myType", wo.contentControl) .build(); <demo-app.my-type model="'Hello wipeout!'"> <set-template> <wo.content-control model="$context.find(app.myType).model"> <set-template> Binding to a specific ancestor: <span wo-content="$model"></span> </set-template> </wo.content-control> </set-template> </demo-app.my-type> <wo.view template-id="'bind-to-ancestor-script-1'"></wo.view>
