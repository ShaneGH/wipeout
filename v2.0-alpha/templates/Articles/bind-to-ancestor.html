Bind to the property of an ancestor

There is a helper function which can find ancestors based on their proerties, type, depth or any other custom attribute you need.

// create a custom class to distinguish type window.myType = wo.contentControl.extend(function myType() { this._super(); }); <myType model="'Hello wipeout!'"> <set-template> <wo.content-control model="$context.find(myType).model"> <set-template> Binding to a specific ancestor: <span data-bind="text: model"></span> </set-template> </wo.content-control> </set-template> </myType> <wo.view template-id="'bind-to-ancestor-script-1'" />
