Observe the value of the property

If you need to observe the property value for change, you can use the busybody.observe function.

Declaration: <wo.view model--mybinding="$this.model"></wo.view> Binding: wo.bindings.mybinding = function mybinding (viewModel, property, renderContext) { // will observe the model property of the view model for change var dispose = busybody.observe(viewModel, property.name, function (oldValue, newValue) { // Do any binding stuff here }); // remember, an observation needs to be disposed of when the binding is torn down. // to do this, return the dispose object return dispose; };

The second argument to watch tells wipeout to execute the callback immediately, as if the property had just changed.
