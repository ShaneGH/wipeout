Binding to global values

The binding scope also includes the window object, so you can bind to global variables

window.exampleValue = { innerValue: "Hello wipeout!" }; <wo.content-control model="exampleValue.innerValue"> <set-template> Binding to global: <span data-bind="text: model"></span> </set-template> </wo.content-control> <wo.view template-id="'bind-to-global-script-1'" />
