Set a string property

By default properties without a type are parsed as strings. You can also explicitly set invoke the string parser.

<wo.content-control> <my-val constructor="string"> Hello wipeout! </my-val> <template> Binding to element value: <span data-bind="text: myVal"></span> </template> </wo.content-control> <wo.view templateId="'set-string-property-script-1'" />
