Computeds

Computeds are properties, the value of which is defined by the result of a function. If anything within the function changes the computed will re-evaluate and update the property. To learn more about computeds, see the busybody documentation and the function.

// view model 1 var person = wo.viewModel("app.person") .templateId("/person.html") // this method initializes the view model .onInitialized(function () { // create a computed property "fullName" which is // an amalgamation of the first name and last name this.computed("fullName", function () { return this.model.firstName + " " + this.model.lastName; }); }) .build(); // view model 2 wo.viewModel("app.name") // for this view model type we want // any changes to the "model" property // to reflect in the property it is bound to. // "tw" stands for "two way" .binding("model", "tw") .templateId("/name.html") .build();

At the URL "/person.html"

<div> First name: <app.name model="$parent.model.firstName"></app.name> <br /> Last name: <app.name model="$parent.model.lastName"></app.name> <br /> <br /> Full name: <span wo-content="$this.fullName"></span> </div>

And at the URL "/person.html"

<input type="text" wo-on-event="keyup" wo-value="$this.model" />

So the following element in html:

<!-- in order to be observed, the model must be an observable --> <app.person model="busybody.makeObservable({firstName: 'Robert', lastName: 'Paulson'})"></app.person>
