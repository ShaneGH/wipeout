Computeds

Computeds are properties whose values are defined by the result of a function. If anything within the function changes, the computed will re-evaluate and update the property. To learn more about computeds, see the busybody documentation, the function and the the class.

// viewModel 1 var person = wo.viewModel("demoApp.person") .templateId("/person.html") // this method initializes the viewModel .onInitialized(function () { // create a computed property "fullName" which is // an amalgamation of the first name and last name this.computed("fullName", function () { return this.model.firstName + " " + this.model.lastName; }); }) .build(); // viewModel 2 wo.viewModel("demoApp.name") // for this viewModel type we want // any changes to the "model" property // to reflect in the property it is bound to. // "tw" stands for "two way" .binding("model", "tw") .templateId("/name.html") .build();

At the URL "/person.html"

<div> First name: <demo-app.name model="$parent.model.firstName"></demo-app.name> <br /> Last name: <demo-app.name model="$parent.model.lastName"></demo-app.name> <br /> <br /> Full name: <span wo-content="$this.fullName"></span> </div>

And at the URL "/person.html"

<input type="text" wo-on-event="keyup" wo-value="$this.model" />

So the following element in html:

<!-- in order to be observed, the model must be an observable --> <demo-app.person model="busybody.makeObservable({firstName: 'Robert', lastName: 'Paulson'})"></demo-app.person>
