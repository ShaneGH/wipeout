Bindings

Binding values together creates powerful chains of pluggable components which are the backbone of a wipeout application.

// view model 1 var timer = wo.viewModel("app.timer") .templateId("/timer.html") // the onInitialized method is called when the view model is created and it's values are set, but before it is rendered .onInitialized(function () { this.show = false; // toggle show every second var _this = this; this.cancel = setInterval(function () { _this.show = !this.show; }, 1000); }) // the dispose function is called when the view model is disposed of .dispose(function () { clearInterval(this.cancel); }) .build(); // view model 2 wo.viewModel("app.blinker") .templateId("/blinker.html") .build();

At the URL /timer.html

<h1>Blinker!</h1> <app.blinker show="$this.show"> </app.blinker>

And at the URL /blinker.html

<div wo-visible="$this.show" style="width: 10px; height: 10px; background-color: red"></div>

So an app.timer element in html will render:

<h1>Welcome to wipeout</h1> <div> <span>It supports templates</span> </div>
