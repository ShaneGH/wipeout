Bindings

Binding values together creates powerful chains of pluggable components which are the backbone of a wipeout application.

// viewModel 1 wo.viewModel("demoApp.person") .templateId("/person.html") .build(); // viewModel 2 wo.viewModel("demoApp.name") .templateId("/name.html") .build();

At the URL "/person.html"

<div> First name: <demo-app.name model="$parent.model.firstName"></demo-app.name> <br /> Last name: <demo-app.name model="$parent.model.lastName"></demo-app.name> <br /> </div>

And at the URL "/person.html"

<span wo-content="$this.model"></span>

So the following element in html:

<demo-app.person model="{firstName: 'Robert', lastName: 'Paulson'}"></demo-app.person>
