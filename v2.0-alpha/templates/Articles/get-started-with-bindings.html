Bindings

Binding values together creates powerful chains of pluggable components which are the backbone of a wipeout application.

// view model 1 wo.viewModel("app.person") .templateId("/person.html") .build(); // view model 2 wo.viewModel("app.name") .templateId("/name.html") .build();

At the URL "/person.html"

<div> First name: <app.name model="$parent.model.firstName"></app.name> <br /> Last name: <app.name model="$parent.model.lastName"></app.name> <br /> </div>

And at the URL "/person.html"

<span wo-content="$this.model"></span>

So the following element in html:

<app.person model="{firstName: 'Robert', lastName: 'Paulson'}"></app.person>
