Bindings

Binding values together creates powerful chains of pluggable components which are the backbone of a wipeout application.

// view model 1 var timer = wo.viewModel("app.timer") .templateId("/timer.html") // the onInitialized method is called when the view model is created and it's values are set, but before it is rendered .onInitialized(function () { this.show = false; // toggle show every second var _this = this; this.cancel = setInterval(function () { _this.show = !_this.show; }, 1000); }) // the dispose function is called when the view model is disposed of .dispose(function () { clearInterval(this.cancel); }) .build(); // view model 2 wo.viewModel("app.blinker") .templateId("/blinker.html") .build();

At the URL /timer.html

<span>Blinker:</span> <app.blinker is-on="$parent.show"> </app.blinker>

And at the URL /blinker.html

<div wo-visible="$this.isOn" style="width: 10px; height: 10px; background-color: red"></div>

So an &lt;app.timer> element in html will render:

<app.timer></app.timer>
