Setting dynamic values

Taking the example from , let's expand it to include something dynamic.

function helloToTheWorld() { this.howManyTimes = 1; this.toString = function () { var output = []; for (var i = 0; i < this.howManyTimes; i++) { output.push("Hello wipeout (" + i + ")!"); } return output.join(", "); }; }

And let's define the amount of times rendered in the definition.

<hello-to-the-world how-many-times="3"> </hello-to-the-world>

This will ultimately render: Hello wipeout (0)!, Hello wipeout (1)!, Hello wipeout (2)!

Notice, the this value in the view model definition. this refers to the current object, and is also where values set in the template (how-many-times="3") are set.
