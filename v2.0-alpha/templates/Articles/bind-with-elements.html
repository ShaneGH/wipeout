Set proerties using XML elements

To set more complex properties or long strings (such as templates) you can set properties as xml elements rather than xml attributes

<wo.content-control> <myJson constructor="json"> { "text": "Hello wipeout!" } </myJson> <template> Binding to element value: <span data-bind="text: myJson.text"></span> </template> </wo.content-control> <wo.view templateId="'bind-with-elements-script-1'" />
