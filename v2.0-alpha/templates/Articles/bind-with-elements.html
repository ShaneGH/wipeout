Set proerties using XML elements

To set more complex properties or long strings (such as templates) you can set properties as xml elements rather than xml attributes

<wo.content-control> <my-object-1> <js-object> <val parser="s"> Hello Wipeout! </val> </js-object> </my-object-1> <set-template> Binding to element value: <span wo-content="$this.myObject1.val"></span> </set-template> </wo.content-control> <wo.view template-id="'bind-with-elements-script-1'" />

This example, the myObject1 property of the viewModel was set to {val: "Hello Wipeout!"}. The part told the wipeout teplate engine that "Hello Wipeout!" should be interpreted as a string.
