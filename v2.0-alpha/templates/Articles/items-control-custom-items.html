Using custom items control items

With a custom itemsControl you can control the types of items which are created by overriding the function.

var myItemsControl = wo.viewModel("myItemsControl", wo.itemsControl) .build(); myItemsControl.createItem = function(model) { var item = new myViewModelType(); item.model(model); item.templateId(this.itemTemplateId()); return item; }; Usage: <my-items-control items="...">...</my-items-control>
