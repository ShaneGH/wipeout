Add a property binding

When binding a viewModel property within a template, the default binding direction is one way (TODOD link). It may be necessary to change this direction to another binding type (two way (TODOD link), one way to source (TODOD, link) etc...) using the binding method.

wo.viewModel("myNamespace.myViewModel") .binding("myProperty", "tw") .build();

There are a variety of bindings you can use, or you can define your own (TODOD link)
