Set the model from the parent's model

In order to keep clean contextual boundaries, the model property of a child view model can be bound to a property of its parent's model using the keywords $parent.model.

<wo.content-control model="{ value: 'Hello wipeout!' }"> <set-template> <wo.content-control model="$parent.model.value"> <set-template> Binding to a property on parent's model: <span wo-content="$model"></span> </set-template> </wo.content-control> </set-template> </wo.content-control> <wo.view template-id="'bind-to-parents-model-script-1'"></wo.view>

In this example there is a parent view model which has a child view model. The model of the child is bound to a property of the model of the parent. $parent points to the scope of the view model 1 level up
