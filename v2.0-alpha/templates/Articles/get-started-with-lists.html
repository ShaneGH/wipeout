Using lists of items

Lists of items are at the core of all javascript frameworks, and wipeout is no different. For simple objects you can simply render any array to render the sum of it's contents.

<div wo-render="[new Date(), ' Item 2 ', 3]"></div> Will render <wo.view template-id--s="gettingStartedWithLists1"></wo.view>

Alternatively, you will more likely want to create a viewModel with a template for each item. You can do this using an

<ul> <wo.items-control items="[{itemText: 'Item 1'}, {itemText: 'Item 2'}]"> <item-template> <li wo-content="$model.itemText"></li> </item-template> </wo.items-control> </ul> Will render <wo.view template-id--s="gettingStartedWithLists2"></wo.view>
