Complex template properties

As well as binding using view model attributes, you can also bind using view model inner element.

<wo.view value1="$model"> <value2>$model</value2> <value3 value="$model"></value3> <value4> <js-object> <value5>$model</value5> </js-object> </value4> </wo.view>
